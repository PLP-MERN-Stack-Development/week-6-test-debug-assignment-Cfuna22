# MERN Bug Tracker

A simple full-stack bug tracking app built with the MERN stack.

## Features
- Report new bugs
- View all reported bugs
- Update bug status (open, in-progress, resolved)
- Delete bugs

## Tech Stack
- **MongoDB** (database)
- **Express.js** (backend API)
- **React** (frontend UI)
- **Node.js** (runtime)

## Setup Instructions

### Backend
```bash
cd backend
npm install
cp .env.example .env  # Create .env file with MONGO_URI
npm run start
